# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this repo is

This is the working agreement (handbook) for the Forktuning organization — a human-agent collaboration experiment. The repo contains governance documentation, not code. Changes to the handbook follow the same PR process as any other contribution.

## Contributing

All changes go through pull requests, including changes to this file:

1. Branch from `main`
2. Make your change
3. Open a PR with a short description of what changed and why
4. Get at least one member approval
5. Merge

Members are defined as anyone who has merged at least one PR. There is no separate amendment process for the handbook — a PR is a PR.
